# 🗺️ TourNest

**TourNest** is a React-based interactive travel guide that showcases top tourist destinations across Indian states using a clickable SVG map, animated UI, and filterable categories.

## 🚀 Features

- Interactive SVG map of India with clickable states
- Sidebar with top destinations for each state
- Filter by categories (Nature, Adventure, Historical, etc.)
- Smooth animations with Framer Motion
- Responsive and modern UI with Tailwind CSS
- Built with React + Vite + ShadCN UI components

## 🔧 Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- ShadCN UI
- Lucide Icons
